# IoT Sensor Node and Custom PCB

Custom PCB sensor node built around the ESP8266. The board collects sensor data and publishes it over MQTT/Wi-Fi to a cloud dashboard for remote monitoring and visualization.

## What This Project Does

This project covers the full stack of an IoT sensor node: hardware design in KiCad, ESP8266 firmware for MQTT publishing, and a cloud dashboard that receives and displays live telemetry data.

## Technologies Used

- ESP8266 Wi-Fi microcontroller
- KiCad PCB design (schematic, layout, Gerbers)
- MQTT protocol for telemetry
- Wi-Fi connectivity
- Cloud dashboard (data ingestion and visualization)
- C/C++ firmware
